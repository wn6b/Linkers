## 🚀 عن المشروع (About the Project)
موقع **Linkers** هو واجهة رقمية متطورة وحديثة تم تصميمها بمعايير سنة 2026، تعمل كبوابة مركزية لجميع روابط ومنصات المطور وصانع المحتوى "وائل | Wano". يتميز الموقع بتصميم تفاعلي يبدأ بواجهة تحميل تشبه موجه الأوامر (Terminal/Bash)، مما يعكس الهوية البرمجية والاحترافية للمطور.

## ✨ المميزات (Features)
* **واجهة دخول برمجية (Terminal-Init):** شاشة تحميل متحركة `bash -- portfolio-init` تعطي انطباعاً برمجياً دقيقاً.
* **تصميم عصري (Modern UI 2026):** متجاوب بالكامل مع جميع الشاشات (الهواتف، الأجهزة اللوحية، والحواسيب).
* **تجميع شامل للمنصات (8+ Platforms):** يضم روابط يوتيوب، تيك توك، سناب شات، واتساب، ديسكورد، جيتهاب، تيليجرام، وفيسبوك.
* **إحصائيات حية:** يعرض حالة النشاط (24/7) وعدد المنصات والزيارات.
* **أداء فائق:** مبني باستخدام تقنيات الويب الأساسية لضمان سرعة التحميل القصوى والاستجابة الفورية.

## 🛠️ التقنيات المستخدمة (Tech Stack)

## ⚙️ طريقة التشغيل والاستضافة (Setup & Hosting)

بما أن المشروع عبارة عن موقع ويب (Static Website)، طريقة رفعه واستضافته جداً بسيطة ومناسبة للإدارة حتى من الهاتف المحمول:

1. **الرفع على GitHub:**
   - قم بإنشاء مستودع (Repository) جديد.
   - ارفع ملفات الموقع (`index.html`, وملفات الـ `CSS` والـ `JavaScript` الخاصة بك).
   
2. **تفعيل GitHub Pages (للاستضافة):**
   - اذهب إلى إعدادات المستودع (Settings).
   - من القائمة الجانبية اختر **Pages**.
   - تحت قسم `Branch`، اختر `main` أو `master` واضغط **Save**.
   - خلال دقائق، سيكون موقعك متاحاً ومستضافاً بالكامل على رابطك.
